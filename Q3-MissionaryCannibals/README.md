Uninformed Search Algorithms Implementation

Problem Implemented:
Missionaries and Cannibals Problem

Description:
This module implements different uninformed search algorithms to solve the Missionaries and Cannibals problem. The program explores the state space to find a valid sequence of moves that safely transfers all missionaries and cannibals across the river while satisfying the problem constraints.

Algorithms Implemented:
- Breadth First Search (BFS)
- Depth First Search (DFS)
- Depth Limited Search (DLS)
- Iterative Deepening Depth First Search (IDDFS)
- Uniform Cost Search (UCS)
